Rock Paper Scissors Pseudocode

1. User will choose any letter in the alphabet or on the keyboard

2. Computer compares this letter to letters in selected array value

3. Letter is displayed in guessed letters

4. Letter is filled in hangman empty spaces

5. Guess counter goes down

  • If user's counter reaches 0 and word isn't guessed, display you lose.

  • If user loses add to their loss ticket

  • If user fills in all letters, display image

  • If user fills in all letters then add to their win ticker

4. We then add to their score.

  • If user wins then we add one to their wins.

  • If user loses then we add one to their losses.

If user fills in all letters prepare to reset game with new click
